Proyectos Destacados (Entre muchos)

Soar Adaptive Optics Module Project

Desarrollado para el Telescopio SOAR de 4m instalado en Cerro Pachón (Cerro Tololo Interamerican Observatory). Sistema de optica adaptativa, con un espejo deformable con 60 actuadores piezoeléctricos, que permite corregir las aberraciones ópticas introducidas por la atmósfera terrestre, que afectan las observaciones astronómicas. La deformación del frente de ondas es medida por un dispositivo óptico (Shack Hartmann), información que es procesada en tiempo real para generar las señales de control hacia el espejo, corrigiendo estas deformaciones.

Eliminación de Humos Negros Hornos RAF

Mediante el control preciso del flujo de aire y combustible (Petroleo denominado ENAP6), al horno de Refino a Fuego, se logra optimizar el proceso de combustión, disminuyendo la generación de contaminantes. Los humos negros contaminantes se convierten en humos blancos constituídos mayormente por vapor de agua. El proyecto se desarrolló en LabView para el sistema de control e interfaz de usuario, y mediante protocolo FieldBus Foundation para la comunicación entre los diversos dispositivos.


Jefferson Lab Multiple Target Project

Cambiar el blanco (target) el en acelerador de electrones del Jefferson Lab requiere de varias semanas de detención del sistema. Se requiere elevar la temperatura desde niveles criogénicos, eliminar el vacio, esperar niveles de radiación adecuados, reemplazar el blanco, y realizar el proceso inverso para llevarlo nuevamente a las condiciones de operación. Disponer de una cinta móvil con 5 blancos diferentes, creando un dispositivo capaz de operar en campos magnéticos muy intensos, elevada radiación, y temperataturas cercanas al cero absoluto, para intercambiarlos, fue el gran logro de este proyecto.
